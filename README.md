
## simple to-do list server


CRUD implementation using express with mongoDB

### Routes
####
#### ``` GET /todos``` 
return a list of all todos
#### ```POST /todos```
create a new todo
#### ```GET /todos/:id```
get a specific todo by ID
#### ```DELETE /todos/:id```
delete a todo by ID
#### ```PATCH /todos/:id```
update a todo
